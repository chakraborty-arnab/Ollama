# Large Language Model Runner

Get up and running with large language models quickly and easily.

## Installation
### Ollama
[Ollama](https://github.com/ollama/ollama/blob/main/README.md)


## Usage
To run a model, use the following commands:
### LLaMA 3
```
ollama run llama
```
### Phi-3
```
ollama run phi-3
```
